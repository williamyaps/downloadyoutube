Install

To run Williamyapdownloadyoutube.exe only need open DOS Shell.

Williamyapdownloadyoutube [OPTIONS] URL [URL...]

Examples:

Easy extract audio:

1. I want extract audio from Westlife album
Williamyapdownloadyoutube --extract-audio --audio-format mp3 https://www.youtube.com/watch?v=15V0nDxT_wQ&list=RD15V0nDxT_wQ

2. I want download download video with the best format without subtitles
so I need to see, whats the format contains:
Williamyapdownloadyoutube -F https://www.youtube.com/watch?v=ZnOAK04
