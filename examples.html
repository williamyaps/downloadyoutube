Examples

To run Williamyapdownloadyoutube.exe only need open DOS Shell.

Williamyapdownloadyoutube [OPTIONS] URL [URL...]

Examples:

 

Easy extract audio:

I want extract audio from Westlife album, so only download mp3 format
Williamyapdownloadyoutube --extract-audio --audio-format mp3 https://www.youtube.com/watch?v=15V0nDxT_wQ

 

Download video:

1. Before download the video file, we need see whats the format file available.
to see all format file available:

ex: Williamyapdownloadyoutube -F https://www.youtube.com/watch?v=15V0nDxT_wQ

2. Begin download, choose the format you want download

Now I want download the best video format + best audio format to file mp4 format without subtitles
ex: Williamyapdownloadyoutube -f bestvideo[ext=mp4]+bestaudio[ext=m4a] --merge-output-format mp4 https://www.youtube.com/watch?v=15V0nDxT_wQ

 

3. To see whats subtitles can download

ex:  Williamyapdownloadyoutube -F --list-subs https://www.youtube.com/watch?v=15V0nDxT_wQ

4. To download the subtitles

ex: Williamyapdownloadyoutube -f bestvideo[ext=mp4]+bestaudio[ext=m4a] --merge-output-format mp4 --sub-lang en https://www.youtube.com/watch?v=6HmcTysGQng

williamyapdownloadyoutube -f 480p-https http://www.viki.com/videos/1101029v-ode-to-joy-episode-38 --write-sub --sub-format srt --sub-lang id
 

5. To convert the subtiles to srt file from vtt file

Williamyapdownloadyoutube -f bestvideo[ext=mp4]+bestaudio[ext=m4a] --merge-output-format mp4 --write-sub --convert-subtitles srt https://www.youtube.com/watch?v=jZrsFoNuN3A

 

 

6. To verbose mode

Williamyapdownloadyoutube -f http-720 -q -v --call home --dump-pages --write-pages http://www.dailymotion.com/video/x2xi513
